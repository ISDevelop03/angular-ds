# API Reference

Property Type Default Description
Component Inputs
variant 'default' | 'placeholder' | 'initials' 'default' Visual variant of the avatar. default shows an image from src; placeholder shows a generic SVG placeholder; initials shows text from children.
size 'small' | 'normal' | 'large' | 'xlarge' | 'xxlarge' 'normal' Size of the avatar. Maps to fixed width/height classes (e.g. small = 24px, normal = 32px, large = 40px, xlarge = 48px, xxlarge = 56px).
alt string '' Alt text for the avatar image. Used for accessibility when variant is default.
src string (default image URL) Image URL for the avatar. Only used when variant is default.
className string '' Optional additional CSS classes to apply to the avatar container. Useful for custom styling or layout adjustments.
children string undefined Text displayed inside the avatar when variant is initials. Typically one or two letters (e.g. user initials like "AB").
