# API Reference

Property Type Default Description
Component Inputs
balance number 0 The balance amount to display. Can be positive or negative. The component automatically displays a plus icon for positive values and a minus icon for negative values. When isDiscrete is true, the balance is masked. The component only renders when balance !== 0.
currency string undefined Optional currency code (e.g., 'USD', 'EUR', 'GBP', 'JPY') displayed alongside the balance. When provided, appears after the formatted balance amount.
isDiscrete boolean false When true, masks the balance amount by displaying "*******" instead of the actual value. Also hides the icon. Use for privacy when displaying sensitive financial information in public or shared environments.
noIcon boolean false When true, hides the plus/minus icon that normally appears for positive/negative balances. The balance amount is still displayed, but without the visual indicator icon.
variant keyof typeof theme 'default' Visual style variant of the balance formatter. Currently supports 'default'. The variant determines the wrapper and currency styling. The component is designed to be extensible with additional variants.
className string undefined Optional additional CSS classes to apply to the balance formatter wrapper. Useful for custom styling or layout adjustments.
currencyClassName string undefined Optional additional CSS classes to apply specifically to the currency display. Use for custom styling of the currency text separately from the balance amount.
