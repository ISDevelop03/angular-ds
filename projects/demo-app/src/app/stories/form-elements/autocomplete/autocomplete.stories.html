Autocomplete

# API Reference

Property Type Default Description
Component Inputs
items IField[] [] Required. Array of available options to display in the autocomplete dropdown. Each item represents a selectable option. See the IField Interface section below for details.
values IField[] [] Array of selected values displayed as badges. Supports two-way binding with [(values)]. When items are selected or removed, this array is updated accordingly. The component implements ControlValueAccessor, so it can be used with Angular reactive forms.
placeholder string '' Placeholder text displayed in the input field when it is empty. Use to provide context or instructions to users (e.g., 'Country...', 'Search options...').
className string '' Optional additional CSS classes to apply to the autocomplete container. Useful for custom styling or layout adjustments.
Component Outputs (Events)
valuesChange EventEmitter<IField[]> - Emitted when the selected values change (items are added or removed). Provides the updated array of selected IField objects. Use with two-way binding [(values)] or listen to this event to track selection changes.
onSelect EventEmitter<IEventEmitter<IField>> - Emitted when an item is selected or deselected. Provides an object with event, values (current selected array), and field (the item that was selected/deselected, or null if removed). Use to perform actions when selections change or to track individual item selections.

# IField Interface

The IField interface defines the structure for items passed to the items and values input properties.

Property Type Default Description
label string - Required. The display text for the option. Shown in the dropdown list and as the badge label when selected. Used for filtering when the user types in the input field.
value string - Required. Unique identifier for the option. Used to match items between items and values arrays, and must be unique within the items array.
checked boolean undefined Optional flag indicating if the item is currently selected. The component automatically manages this property based on the values array. When an item is in values, its checked flag is set to true in the dropdown.
