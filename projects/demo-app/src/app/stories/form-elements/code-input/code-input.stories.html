# API Reference

Property Type Default Description
Component Inputs
id string - Required. Unique identifier for the input element. Used to generate the final input ID as ds-{{id}}. Also used for form association and accessibility.
name string - Required. HTML name attribute for the input element. Used for form submission and grouping inputs. When multiple inputs share the same name, they can be grouped together.
value any 0 The input value. Can be a string or number. Supports two-way binding. The component is optimized for numeric code inputs (e.g., verification codes, PINs).
type string 'text' HTML input type attribute. The component template uses type="number" by default, but this property can be used to override it. Common values: 'text', 'number', 'tel'.
placeholder string undefined Optional placeholder text displayed in the input when it is empty. Use to provide context or instructions to users (e.g., 'Enter code', '0000').
disabled boolean false When true, disables the input, preventing user interaction. The input appears grayed out with reduced opacity and cannot be edited.
required boolean false When true, marks the input as required for form validation. Browsers will prevent form submission if the input is empty.
hasError boolean false When true, indicates that the input has a validation error. Use with errorMessage to display error feedback to users.
errorMessage string undefined Optional error message text displayed when hasError is true. Use to provide validation feedback to users.
label string undefined Optional text label displayed above or alongside the input. Use for accessibility and user guidance.
description string undefined Optional descriptive text displayed below the input. Use to provide additional context or instructions to users.
prefix string undefined Optional prefix text or icon displayed before the input value. Use for currency symbols, units, or other visual indicators.
suffix string undefined Optional suffix text or icon displayed after the input value. Use for currency symbols, units, or other visual indicators.
className string '' Optional additional CSS classes to apply to the input element. Useful for custom styling or layout adjustments. The component automatically forwards additional HTML attributes (like maxlength, minlength, pattern) to the native input element.
Component Outputs (Events)
onInput EventEmitter<Event> - Emitted when the input value changes (on every keystroke). Provides the input event object. Use to handle real-time value changes and update the value property.
onChange EventEmitter<Event> - Emitted when the input value changes and the input loses focus. Provides the change event object. Use to handle value changes after the user finishes editing.
onClick EventEmitter<Event> - Emitted when the input is clicked. Provides the click event object. Use for custom click handling.
onFocus EventEmitter<FocusEvent> - Emitted when the input receives focus. Provides the focus event object. Use to handle focus events and perform actions when the input is focused.
onBlur EventEmitter<FocusEvent> - Emitted when the input loses focus. Provides the blur event object. Use to handle blur events and perform validation or cleanup when the input loses focus.
onKeydown EventEmitter<KeyboardEvent> - Emitted when a key is pressed down. Provides the keyboard event object. Use to handle keyboard interactions and shortcuts.
onKeyup EventEmitter<KeyboardEvent> - Emitted when a key is released. Provides the keyboard event object. Use to handle keyboard interactions.
onEnter EventEmitter<KeyboardEvent> - Emitted when the Enter key is pressed. Provides the keyboard event object. Use to handle form submission or trigger actions when Enter is pressed.
onEscape EventEmitter<KeyboardEvent> - Emitted when the Escape key is pressed. Provides the keyboard event object. Use to handle cancellation or close actions when Escape is pressed.
onPaste EventEmitter<ClipboardEvent> - Emitted when content is pasted into the input. Provides the clipboard event object. Use to handle paste events and validate or transform pasted content.
onSelect EventEmitter<UIEvent> - Emitted when text in the input is selected. Provides the UI event object. Use to handle text selection events.
