Todo Tab

TodoTab

# API Reference

Property Type Default Description
Component Inputs
title string '' Required. The title text displayed below the count and icon. Shown in small text (0.75rem / 12px) with medium font weight. This is the label for the todo tab (e.g., "Mes opérations en attente", "Tâches administratives").
count number 0 The number displayed prominently at the top of the tab. Shown in large text (2rem / 32px) with bold font weight and gray color. This typically represents the number of items or tasks in the category (e.g., 6, 4). The count is always displayed, even if 0.
icon string '' Optional icon identifier displayed next to the count. Should match an available icon in your icon library (e.g., 'flag', 'message', 'file'). The icon is displayed in red color (text-red-500) with a size of 20px. The icon is only rendered if provided (conditional rendering with *ngIf).
isOpen boolean false When true, indicates that the tab is in an open/active state. Open tabs have:
  • Red bottom border: border-red-500
  • Arrow icon rotated 180 degrees and colored red
When false (closed), the tab has:
  • Transparent bottom border: border-transparent
  • Arrow icon in default position and color
Use to control the visual state of the tab and indicate which section is currently active or expanded.
onClick () => void undefined Optional callback function executed when the tab is clicked. The function receives no parameters. Use to handle tab clicks, toggle the isOpen state, or perform other actions when the user interacts with the tab. The callback is only executed if provided (conditional execution with onClick && onClick()).
className string '' Optional additional CSS classes to apply to the tab container. Useful for custom styling, spacing, or layout adjustments. The component has default styling with flex layout, padding, gap spacing, and conditional borders based on isOpen.
