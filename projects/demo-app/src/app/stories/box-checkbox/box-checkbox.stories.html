# API Reference

Property Type Default Description
Component Inputs
checked boolean false The checked state of the checkbox. When true, the checkbox is checked. When false, it is unchecked. Supports two-way binding.
label string undefined Optional text label displayed next to the checkbox. When provided, appears to the right of the checkbox input.
disabled boolean false When true, disables the checkbox, preventing user interaction. The checkbox appears grayed out and cannot be toggled.
tooltip string | TemplateRef<any> undefined Optional tooltip content displayed when hovering over the info icon. Can be a string for simple text tooltips or a TemplateRef for complex, reusable tooltip content with custom components. When provided, displays an info icon on the right side of the box.
change () => void () => {} Callback function executed when the checkbox state changes. Use to handle checkbox toggle events and update the checked state or perform other actions when the checkbox is clicked.
className string '' Optional additional CSS classes to apply to the box checkbox container. Useful for custom styling or layout adjustments. The container has a border and padding by default.
