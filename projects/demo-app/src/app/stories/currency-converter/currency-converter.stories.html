# API Reference

Property Type Default Description
Component Inputs
value number - Required. The numeric amount value displayed in the input field. Supports two-way binding with [(value)]. When the user types in the input, this value is updated and the valueChange event is emitted.
currency string - Required. The currently selected currency code (e.g., 'EUR', 'USD', 'MAD'). This should match the value property of one of the items in the currencies array. Supports two-way binding with [(currency)].
currencies CurrencyProps[] [] Required. Array of available currency options to display in the currency selector dropdown. Each currency object defines a currency option with its code, label, symbol, color, and flag image. See the CurrencyProps Interface section below for details.
Component Outputs (Events)
valueChange EventEmitter<number> - Emitted when the numeric input value changes (on every keystroke). Provides the new numeric value. Use with two-way binding [(value)] or listen to this event to track value changes. The component automatically converts the input string to a number.
currencyChange EventEmitter<any> - Emitted when the currency selection changes. Provides the selected currency object (with value, label, symbol, color, image properties). Use with two-way binding [(currency)] or listen to this event to track currency changes. Access the currency code via $event.value.

# CurrencyProps Interface

The CurrencyProps interface defines the structure for currency items passed to the currencies input property.

Property Type Default Description
value string - Required. The currency code (e.g., 'EUR', 'USD', 'MAD'). This is the unique identifier for the currency and should match the currency input value when this currency is selected.
label string - Required. The display name of the currency (e.g., 'Euro', 'Dollar', 'Dirham'). Shown in the currency selector dropdown.
symbol string - Required. The currency symbol (e.g., '€', '$', 'د.م'). Used for display purposes in the currency selector.
color string - Required. The color associated with the currency. Can be a CSS color value (e.g., 'red', '#FF0000', 'rgb(255, 0, 0)'). Used for styling the currency selector.
image string - Required. URL or path to the currency flag image. Displayed in the currency selector to provide visual identification of the currency. Should be a valid image URL or path.
