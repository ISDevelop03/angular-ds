# API Reference

Property Type Default Description
Component Inputs
pages Page[] [] Required. Array of page objects representing the breadcrumb trail. Each page object defines a navigation level. See the Page Interface section below for details.
homeUrl string '' Optional URL for the home link. When provided, displays a home icon or "Accueil" text at the beginning of the breadcrumb. When showLimit is true, shows only the home icon instead of text.
showLimit boolean false When true, limits the number of visible breadcrumb items. If homeUrl is provided and there are more than 2 pages, shows only the first page. Otherwise, if there are more than 3 pages, shows only the first 2 pages. The remaining pages are collapsed with "..." and the last page is always shown.
variant keyof typeof breadcrumb 'default' Visual style variant of the breadcrumb. Currently supports 'default'. The variant determines the wrapper, list, link, and separator styling. The component is designed to be extensible with additional variants.
prevUrl string '' Optional URL for the previous navigation link. When provided, displays a left arrow icon before the breadcrumb list that links to the previous page.
prevClassName string '' Optional CSS classes to apply to the previous navigation link (left arrow). Use for custom styling of the previous button.
nextUrl string '' Optional URL for the next navigation link. When provided, displays a right arrow icon after the breadcrumb list that links to the next page.
nextClassName string '' Optional CSS classes to apply to the next navigation link (right arrow). Use for custom styling of the next button.
className string '' Optional additional CSS classes to apply to the breadcrumb wrapper. Useful for custom styling or layout adjustments.

# Page Interface

The Page interface defines the structure for page objects passed to the pages input property.

Property Type Default Description
id string - Required. Unique identifier for the page. Used internally to track and manage breadcrumb items.
name string - Required. The display text for the breadcrumb item. This is the visible label shown in the breadcrumb trail.
href string undefined Optional URL or route path for the page. When provided, the breadcrumb item becomes a clickable link. When not provided, uses '#.' as a fallback. The last page in the list is always rendered as a non-clickable div, regardless of this value.
current boolean undefined Optional flag indicating if this is the current page. When true, applies active styling (bold text, darker color) and sets aria-current="page" for accessibility. Use to highlight the current page in the breadcrumb trail.
