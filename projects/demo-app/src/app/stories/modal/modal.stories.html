# API Reference

Property Type Default Description
Component Inputs
isShown boolean false Controls visibility. Set to true to open the modal, false to close it.
type 'success' | 'warning' | 'error' | 'info' 'info' Semantic type; affects icon background and color. Use with icon for consistent styling (e.g. info + info-circle, warning/error + danger, success + check).
icon string '' Icon id for ds-icon (e.g. info-circle, danger, check). Omit or leave empty to hide the icon.
title string — Modal heading text.
description string — Body text below the title.
mainAction string — Label for the primary button. Button is hidden if not set.
secondaryAction string — Label for the secondary button. Button is hidden if not set.
className string '' Optional CSS classes applied to the modal root element.
Component Outputs
close EventEmitter<void> — Emitted when the modal is closed (close button, backdrop click, or Escape key). Bind to set isShown to false.
mainActionOnClick EventEmitter<void> — Emitted when the primary action button is clicked.
secondaryActionOnClick EventEmitter<void> — Emitted when the secondary action button is clicked.
