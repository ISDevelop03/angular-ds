# API Reference

Property Type Default Description
Component Inputs
label string undefined Optional label text displayed above the input. Use to provide context or instructions to users.
tooltip string undefined Optional tooltip content. When set, an info icon is shown next to the label; hovering displays the tooltip.
required boolean undefined When true, a red asterisk is displayed next to the label to indicate the field is required.
hasError boolean false When true, applies error styling to the country select and phone input. Use with errorMessage to show validation feedback.
errorMessage string undefined Error message displayed below the input when hasError is true. Use to provide specific validation feedback.
description string undefined Optional helper text displayed below the input. Use for instructions or additional context (e.g., "Please enter your mobile phone number...").
countriesPrefixes Record<string, string>[] [] Required. Array of country prefix options for the select. Each item should have value and label (e.g., { value: '+212', label: '+212' }).
selectedCountryPrefix string '+212' The currently selected country calling code (e.g., '+212', '+33'). Must match one of the value entries in countriesPrefixes.
placeholder string 'Enter your phone number' Placeholder text displayed in the phone number input when empty.
className string '' Optional CSS classes applied to the component root. Use for custom styling or layout.
