Illustration

# API Reference

Property Type Default Description
Component Inputs
src string '' The source URL or path to the illustration image. This is the primary content displayed by the component. Supports relative paths (e.g., '../assets/illustration.png') or absolute URLs.
size 'lg' | 'md' | 'sm' | 'xs' 'lg' Controls the dimensions of the illustration. Available sizes and their dimensions:
  • lg: 300×224 px
  • md: 200×126 px
  • sm: 170×127 px
  • xs: 120×90 px
className string '' Optional additional CSS classes to apply to the illustration container. Useful for custom styling, positioning, or layout adjustments.
