MatriceSignatureCard

# API Reference

Property Type Default Description
Component Inputs
className string '' Optional CSS classes merged onto the root container.
items AccordionItem<MatriceSignatureCardContext>[] [] Accordion sections for the signature matrix. Pass items without content: the component assigns an internal ng-template that renders group list, users, validator count, and order from context. See MatriceSignatureCardContext below.

This component has no outputs; interaction is handled by the nested ds-accordion.

# MatriceSignatureCardContext

Each accordion item’s context must match this shape. Template variables groupList, userList, validatorNumber, and order are bound from it.

Property Type Default Description
groupList string[] - Group names shown in the “Liste des groupes” summary (comma-separated in the UI).
userList string[] - User labels rendered as small badges in the second group list block.
validatorNumber number - Value shown for “Nombre de validateurs”.
order string - Text shown for “Ordre de signature” (e.g. '1er').
