InputSearch Component

Basic Input Search

A compact search input that expands when the search icon is clicked. Shows a clear button when the field has text.

With Custom Max Width

Control the maximum width of the expanded input. Useful for fitting different layouts.

With Custom Placeholder

Override the default "Search" placeholder with custom text.

With Initial Value

Pre-populate the search field with an initial value using the value input.

# API Reference

Property Type Default Description
Component Inputs
value string '' The current search text. Supports two-way data binding with [(value)]. When the user types or clears the field, this value updates. If the field is cleared and the user clicks outside, the input collapses (when empty).
placeholder string 'Search' Placeholder text displayed in the input when it is empty. Shown when the search is expanded.
maxWidth string '280px' Maximum width of the expanded input area. Accepts any valid CSS width value (e.g. '280px', '50%', '100%'). The input animates to this width when expanded.
className string '' Optional CSS classes applied to the component root wrapper. Use for layout (e.g. w-full, max-w-md) or custom styling.

# Behavior

  • Expand: Click the search icon to expand the input. The input animates to maxWidth.
  • Collapse: When the input is empty and the user clicks outside the component, the input collapses back to icon-only.
  • Clear: A clear (X) button appears when there is text; clicking it clears the value.
