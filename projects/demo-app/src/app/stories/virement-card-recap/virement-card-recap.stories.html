# API Reference

Property Type Default Description
Component Inputs
className string '' Optional CSS classes applied to the component root element.
items TRecapItems[] [] Array of recap items to display. See the TRecapItems interface section below for details.

# TTransferAmount Interface

The TTransferAmount type defines the shape of the amount property within each TRecapItems entry.

Property Type Default Description
amount number - The numeric amount of the transfer.
currency string - Currency code (e.g. 'EUR', 'MAD').

# TRecapItems Interface

The TRecapItems interface defines the structure for recap items passed to the items input property.

Property Type Default Description
beneficiaryName string - Name of the transfer beneficiary.
beneficiaryAccountNumber string - Account number of the beneficiary.
motif string - Transfer reason or label.
amount TTransferAmount - Transfer amount and currency. See the TTransferAmount interface above.
