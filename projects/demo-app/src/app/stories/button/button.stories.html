# API Reference

Property Type Default Description
Component Inputs
variant keyof typeof defaultTheme.variants 'primary' Visual style variant of the button. Available options: 'primary', 'secondary', 'gray', 'outline'. Each variant has distinct colors and styling. When outline is true, applies outline styling to the selected variant.
size keyof typeof defaultTheme.size 'normal' Size of the button. Available options: 'small', 'normal', 'medium', 'large'. Controls padding and text size. Icon-only buttons use different padding than text buttons.
disabled boolean false When true, disables the button, preventing user interaction. The button appears grayed out and cannot be clicked. Note: A link button (when href is provided) cannot be disabled and will throw an error if disabled is set to true.
outline boolean false When true, applies outline styling to the button variant (transparent background with border). When false, uses filled background styling.
pill boolean false When true, applies fully rounded corners (pill shape) to the button. When false, uses the default rounded corners.
icon string undefined Icon identifier for icon-only buttons. When provided, displays only the icon (no text content). Should match an available icon in your icon library (e.g., 'arrow-right', 'check', 'close'). Mutually exclusive with iconLeft and iconRight.
iconLeft string undefined Icon identifier displayed on the left side of the button text. Should match an available icon in your icon library. Mutually exclusive with icon. Use with button text content.
iconRight string undefined Icon identifier displayed on the right side of the button text. Should match an available icon in your icon library. Mutually exclusive with icon. Use with button text content.
isLoading boolean false When true, displays a loading spinner icon (Loader) with animation. Use to indicate that an action is in progress. The spinner appears before the button content.
href string undefined Optional URL or route path. When provided, the button renders as an anchor tag (<a>) instead of a button element, making it navigable. Link buttons cannot be disabled. Use for buttons that should navigate to other pages or resources.
target string null Optional target attribute for link buttons (when href is provided). Common values: '_blank' (open in new tab), '_self' (open in same tab). Only applies when href is set.
type string 'button' HTML button type attribute. Available options: 'button', 'submit', 'reset'. Only applies when href is not provided (button element, not anchor). Use 'submit' for form submission buttons.
className string '' Optional additional CSS classes to apply to the button. Useful for custom styling or layout adjustments.
style string '' Optional inline CSS styles to apply to the button. Accepts a string of CSS style properties (e.g., 'color: red; width: 100px;'). Use for dynamic styling that cannot be achieved with classes.
theme any undefined Optional custom theme object to override the default button theme. When provided, uses this theme instead of the default. When not provided, falls back to defaultTheme. Use for completely custom button styling.
Component Outputs (Events)
onClick EventEmitter<any> - Emitted when the button is clicked. Provides the click event object. Only fires for button elements (not link buttons when href is provided). Use for handling button click events and performing actions.
