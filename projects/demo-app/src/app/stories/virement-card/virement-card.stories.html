VirementCard

# API Reference

Property Type Default Description
Component Inputs
className string '' Optional CSS classes applied to the card root element.
beneficiaryName string - Name of the transfer beneficiary.
beneficiaryAccountNumber string - Account number of the beneficiary.
motif string - Transfer reason or label (editable in edit mode).
amount TTransferAmount - Transfer amount and currency. See the TTransferAmount interface section below for details.
Component Outputs (Events)
onSave EventEmitter<TTransfer> - Emitted when the user saves in edit mode. See the TTransfer interface section below for the payload shape.
onDelete EventEmitter<void> - Emitted when the user triggers delete (e.g. delete button).

# TTransferAmount Interface

The TTransferAmount type defines the shape for the amount input property.

Property Type Default Description
amount number - The numeric amount of the transfer.
currency string - Currency code (e.g. 'EUR', 'MAD').

# TTransfer Interface

The TTransfer type defines the payload emitted by the onSave event.

Property Type Default Description
motif string - The transfer reason or label as entered by the user.
amount number - The numeric amount as entered by the user.
