Invoice Card

# API Reference

Property Type Default Description
Component Inputs
title string '' The title text displayed in the invoice card. Shown in bold text below the image. This is the primary heading for the invoice (e.g., 'INVOICE 1').
image string '' Required. URL or path to the invoice image. The image is displayed on the left side of the card in a bordered container with rounded corners. The image is used as the alt text for accessibility.
items InvoiceActions[] [] Optional array of action items displayed as a dropdown menu. When provided and the array has items, a dropdown with an arrow-down icon is shown on the right side of the card. Each item can have a label, icon, href, and onClick handler. See the InvoiceActions Interface section below for details. If both items and actions are empty, an arrow-right icon is displayed instead.
actions ICallToActionIcon[] [] Optional array of action icons displayed on the right side of the card. When provided and the array has items, call-to-action icons are shown using the ds-call-to-action-icons component with 'borderless' variant. Actions are displayed when items is empty or has no items. See the ICallToActionIcon Interface section below for details.
href string '' Optional URL or route path. When provided, the entire card is wrapped in an anchor tag, making it clickable for navigation. The card click handler still fires when the card is clicked. Can be a relative path, absolute URL, or route path depending on your routing setup. When not provided, the card is rendered as a div with cursor-pointer styling.
id any '' Optional unique identifier for the invoice card. Use for tracking, identification, or DOM manipulation. Currently available as an input property for future use or custom implementations.
data any undefined Optional data object passed to action handlers and the onClick event. When action items or action icons are clicked, this data is included in the event payload. The component automatically wraps action handlers to include this data. Use to pass additional context or invoice information to handlers.
className string '' Optional additional CSS classes to apply to the invoice card container. Useful for custom styling, layout adjustments, or responsive sizing (e.g., max-w-[325px]). The card has default styling with background, padding, border radius, and flex layout.
Component Outputs (Events)
onClick EventEmitter<any> - Emitted when the invoice card is clicked. Provides an object with event and data properties: { event: Event, data: any }. The event is the click event, and data comes from the component's data input property. The entire card is clickable. Use to handle card selection, navigation, or invoice details display.

# InvoiceActions Interface

The InvoiceActions interface defines the structure for action items passed to the items input property. These items are displayed in a dropdown menu.

Property Type Default Description
label string - Required. The label text displayed in the dropdown menu item. This is the visible text for the action option (e.g., 'View', 'Download', 'Delete').
icon string - Required. Icon identifier displayed next to the label in the dropdown menu item. Should match an available icon in your icon library (e.g., 'eye', 'download', 'trash').
href string - Required. URL or route path for navigation. When the dropdown item is clicked, this href is used for navigation. Can be a relative path, absolute URL, or route path depending on your routing setup.
onClick (data: any) => void - Required. Callback function executed when the dropdown item is clicked. Receives the data object from the component's data input property. The component automatically wraps this handler to pass the data. Use to handle custom actions, API calls, or additional logic beyond navigation.

# ICallToActionIcon Interface

The ICallToActionIcon interface defines the structure for action icons passed to the actions input property. These icons are displayed using the ds-call-to-action-icons component.

Property Type Default Description
icon string - Required. Icon identifier displayed in the action button. Should match an available icon in your icon library (e.g., 'more-1', 'edit', 'delete').
onClick (item?: any) => void undefined Optional callback function executed when the action icon is clicked. The component automatically wraps this handler to pass an object with event and data properties: { event: Event, data: any }. Use to handle icon clicks, trigger actions, or open modals.
buttonClassName string undefined Optional additional CSS classes to apply to the action button element. Useful for custom styling or layout adjustments specific to individual action icons.
dropdownItems DropdownItem[] undefined Optional array of dropdown items. When provided, the action icon becomes a dropdown trigger, displaying a menu when clicked. Each item can have a label, icon, and onClick handler. Use for actions that require multiple options or sub-actions.
