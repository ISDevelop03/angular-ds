Inbox Card

# API Reference

Property Type Default Description
Component Inputs
title string undefined Optional title text displayed at the top of the card content. Shown in semibold text. This is the primary heading for the inbox notification (e.g., 'Update Completed', 'Urgent Notice').
content string undefined Optional content text displayed below the title. Shown in smaller, medium-weight text. Use to provide additional details or the main message of the notification (e.g., 'Your update was successful.', 'This inbox is unread.').
date string undefined Optional date or time text displayed at the bottom of the card content. Shown with a clock icon. Use to indicate when the notification was received or created (e.g., '2 days ago', '5 mins ago').
isRead boolean false When false (unread), applies distinct styling: red left border (border-l-red-500) and light background (bg-neutral-50). When true (read), uses transparent left border and default background. Use to visually distinguish between read and unread notifications.
link string undefined Optional URL or route path. When provided, renders an invisible anchor tag covering the entire card, making the entire card clickable for navigation. The link is accessible via screen readers. Can be a relative path, absolute URL, or route path depending on your routing setup.
id string undefined Optional unique identifier for the inbox card. Use for tracking, identification, or DOM manipulation. Currently available as an input property for future use or custom implementations.
data any undefined Optional data object passed to the onClick event handler. When the card is clicked, the handler receives an object with event and data properties: { event: Event, data: any }. Use to pass additional context or information to the click handler.
Component Outputs (Events)
onClick EventEmitter<any> - Emitted when the inbox card is clicked. Provides an object with event and data properties: { event: Event, data: any }. The event is the click event, and data comes from the component's data input property. The entire card is clickable. Use to handle card selection, mark as read, or navigate to details.
