Error Card

# API Reference

Property Type Default Description
Component Inputs
items IErrorCard[] [] Required. Array of error items to display in the card. Each item represents an error or validation message with a label and optional value. Items are displayed as a list with red error styling. See the IErrorCard Interface section below for details.
className string '' Optional additional CSS classes to apply to the error card container. Useful for custom styling or layout adjustments (e.g., max-w-[342px] for width control). The card has default red border and background styling.

# IErrorCard Interface

The IErrorCard interface defines the structure for error items passed to the items input property.

Property Type Default Description
label string - Required. The error label or field name displayed on the left side of the error item. Shown in medium font weight. This typically identifies the field or area where the error occurred.
value string undefined Optional error message or value displayed on the right side of the error item. When provided, the item layout uses justify-between to space the label and value. When not provided, the item uses justify-start and only shows the label.
className string undefined Optional CSS classes to apply to the individual error item (list item). Use for custom styling of specific error items. When provided, also applies additional text sizing to the span element.
