# API Reference

Property Type Default Description
Component Inputs
title string '' The title text displayed in the card. Shown in bold text between the bank icon and the arrow icon. When not provided or empty, the title is not displayed. This is the primary content of the holding card.
href string '' Optional URL or route path. When provided, the card renders as an anchor tag (<a>) making it a clickable link. When not provided or empty, the card renders as a <div> element. Use for navigation to holding details or related pages. Can be a relative path, absolute URL, or route path depending on your routing setup.
className string '' Optional additional CSS classes to apply to the holding card container. Useful for custom styling or layout adjustments. The card has default styling with border, background, padding, hover effects (shadow on hover), and smooth transitions.
