Display Price Card

# API Reference

Property Type Default Description
Component Inputs
text string '' The label text displayed before the price (e.g., 'Montant total de la sélection :'). Shown in bold on the left side of the price section.
price string '' The price value displayed on the right side (e.g., '730 MAD'). Can include currency symbols or codes. Displayed in large, bold text alongside the icon.
icon string 'receipt-lines' Icon identifier displayed next to the price. Should match an available icon in your icon library. Defaults to 'receipt-lines'. The icon appears in a colored style (crame-500) with a size of 24px.
items DisplayPriceCardItem[] [] Optional array of detail items displayed above the price section. Each item shows a label on the left and a value on the right. When provided, a dashed border separator appears above the price section. See the DisplayPriceCardItem Interface section below for details.
buttons ButtonItem[] [] Optional array of action buttons displayed at the bottom of the card. Buttons are aligned to the right. Each button can have custom styling, icons, and click handlers. See the ButtonItem Interface section below for details.
className string '' Optional additional CSS classes to apply to the card container. Useful for custom styling or layout adjustments (e.g., max-w-[600px] for width control).
data any undefined Optional data object passed to button click handlers. When a button is clicked, the handler receives an object with event and data properties: { event: $event, data: data }. Use to pass context or additional information to button handlers.

# DisplayPriceCardItem Interface

The DisplayPriceCardItem interface defines the structure for detail items passed to the items input property.

Property Type Default Description
label string - Required. The label text displayed on the left side of the item (e.g., 'Subtotal', 'Tax', 'Discount'). Shown in regular weight text.
value string - Required. The value text displayed on the right side of the item (e.g., '500 MAD', '50.00', '10%'). Shown in bold text.
showLine boolean undefined Optional flag indicating whether to display a dashed line between the label and value. When true, displays a horizontal dashed border line connecting the label and value. Use for visual separation when the label and value are far apart.

# ButtonItem Interface

The ButtonItem interface defines the structure for action buttons passed to the buttons input property.

Property Type Default Description
text string - Required. The button label text displayed on the button.
className string - Required. CSS classes to apply to the button. Useful for custom styling or layout adjustments.
onClick (event: Event) => void - Required. Callback function executed when the button is clicked. Receives an object with event and data properties: { event: Event, data: any }. The data comes from the component's data input property.
variant string 'primary' Required. Button style variant. Defaults to 'primary' if not provided. Available options: 'primary', 'secondary', 'gray', 'outline'.
size string 'normal' Required. Button size. Defaults to 'normal' if not provided. Available options: 'small', 'normal', 'medium', 'large'.
type 'submit' | 'button' | 'reset' 'button' Required. HTML button type attribute. Defaults to 'button' if not provided. Use 'submit' for form submission buttons.
isLoading boolean - Required. When true, displays a loading spinner on the button. Use to indicate that an action is in progress.
icon string - Required. Icon identifier for icon-only buttons. Should match an available icon in your icon library. Mutually exclusive with iconLeft and iconRight.
iconRight string - Required. Icon identifier displayed on the right side of the button text. Should match an available icon in your icon library. Mutually exclusive with icon.
iconLeft string - Required. Icon identifier displayed on the left side of the button text. Should match an available icon in your icon library. Mutually exclusive with icon.
