# API Reference

Property Type Default Description
Component Inputs
title string - Optional title text displayed in the card. Shown in small text (0.75rem / 12px) with bold font weight. This is typically a short label or name for the item (e.g., "Bitcoin", "Ethereum", "Stock Name"). The title is only rendered if provided (conditional rendering with *ngIf). Also used as the alt text for the image when provided.
image string - Optional URL path to an image displayed on the left side of the card. The image is displayed as a circular 24px Ã— 24px (size-6) image with object-cover to maintain aspect ratio. The image is only rendered if provided (conditional rendering with *ngIf). Use for logos, icons, or thumbnails (e.g., cryptocurrency logos, stock symbols, product images).
price number - Optional numeric value displayed below the title. Shown in small text (0.75rem / 12px) with neutral-700 color. This is typically a price, value, or numeric metric (e.g., 50000, 1234.56, 99.99). The price is only rendered if provided (conditional rendering with *ngIf). You can format the price before passing it to the component using Angular pipes (e.g., currency, number formatting).
className string undefined Optional additional CSS classes to apply to the ticker card container. Useful for custom styling, spacing, or layout adjustments. The component has default styling with flex layout, centered items, and gap spacing (gap-2).
