GroupSignataireCheckbox

# API Reference

Property Type Default Description
Component Inputs
className string '' Optional CSS classes merged onto the root container.
users string[] [] Display names for signatories. Initials avatars and the user list in the dropdown are derived from these strings.
checked boolean false Whether the group checkbox is checked. Updates internal state when this input changes.
Component Outputs (Events)
change EventEmitter<boolean> - Emitted when the user toggles the checkbox. Payload is the new checked value.
toggle EventEmitter<boolean> - Emitted when the users dropdown opens or closes. Payload reflects the dropdown open state from ds-dropdown-holder.
