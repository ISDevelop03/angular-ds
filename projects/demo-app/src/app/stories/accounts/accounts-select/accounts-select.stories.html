Accounts Select

# API Reference

Property Type Default Description
Component Inputs
items SelectItem[] [] Required. Array of account items to display in the dropdown. Each item represents an account option. See the SelectItem Interface section below for details.
value string | null null The selected value(s). When multiple is true, this should be an array of strings. When multiple is false, this is a single string or null. Supports two-way binding with [(value)].
label string undefined Optional label text displayed above the select component. Use to provide context or instructions to users.
placeholder string 'Select an option' Placeholder text displayed when no option is selected. Shown in the trigger button when value is null.
variant keyof typeof theme 'default' Visual style variant of the select component. Determines the appearance and styling of the component.
disabled boolean false When true, disables the select component, preventing user interaction. The component appears grayed out and cannot be opened.
hasError boolean false When true, applies error styling to indicate a validation error. Use in conjunction with errorMessage to display validation feedback.
errorMessage string undefined Optional error message displayed below the select component when hasError is true. Use to provide specific validation feedback to users.
emptyLabel string 'No options' Text displayed in the dropdown when the items array is empty. Use to inform users that no options are available.
multiple boolean false When true, enables multiple selection. The value should be an array of strings, and users can select multiple account options.
isLoading boolean false When true, displays a loading state, preventing user interaction while data is being fetched. Use during async operations to indicate that options are being loaded.
buttonIcon string undefined Optional icon identifier displayed in the trigger button. Should match an available icon in your icon library (e.g., 'message', 'account', 'wallet').
customOptionsStyles { [key: string]: any } undefined Optional inline styles object to apply to the options dropdown container. Keys are CSS property names. Use for custom positioning or styling of the dropdown menu.
className string undefined Optional additional CSS classes to apply to the select container. Useful for custom styling or layout adjustments.
Component Outputs (Events)
valueChange EventEmitter<string | string[]> - Emitted when the selected value changes. Provides the selected value (string for single selection, string array for multiple selection). Use with two-way binding [(value)] or listen to this event to track selection changes.

# SelectItem Interface

The SelectItem interface defines the structure for account items passed to the items input property.

Property Type Default Description
value string - Required. Unique identifier for the account item. Used to match against the selected value and must be unique within the items array.
icon string - Required. Icon identifier or name displayed for the account. Should match an available icon in your icon library.
accountName string - Required. The display name of the account. Shown as the primary text label for the account option.
accountNumber string - Required. The account number or identifier displayed alongside the account name. Used to uniquely identify the account.
soldeComptable Solde - Required. The accounting balance (book balance) for the account. See the Solde Interface section below for details.
soldeTempsReel Solde - Required. The real-time balance for the account. Represents the current available balance including pending transactions. See the Solde Interface section below for details.
unavailable boolean undefined Optional flag indicating if the account is unavailable for selection. When true, the option may be displayed but disabled or styled differently to indicate it cannot be selected.

# Solde Interface

The Solde interface defines the structure for balance information used in SelectItem.

Property Type Default Description
value number - Required. The numeric balance amount. Represents the balance value for the account.
currency string - Required. The currency code (e.g., 'USD', 'EUR', 'GBP') for the balance. Used for formatting and display purposes.
currency_flag string - Required. The currency flag identifier or emoji code. Used to display a visual flag or symbol representing the currency alongside the balance.
