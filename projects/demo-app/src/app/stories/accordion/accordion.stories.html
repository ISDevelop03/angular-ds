# API Reference

Property Type Default Description
Component Inputs
variant 'default' | 'borderless' 'default' Visual style variant of the accordion. 'default' for standard styling with borders and background, 'borderless' for a cleaner look without borders.
items AccordionItem[] [] Array of accordion items to display. Each item represents a collapsible panel. See the AccordionItem Interface section below for details.
allowMultiple boolean false When true, allows multiple accordion panels to be open simultaneously. When false, opening one panel automatically closes all others (accordion behavior).
textClassName string '' Optional CSS classes to apply to the accordion item title text. Use for custom text styling like font size, weight, or color.
className string '' Optional additional CSS classes to apply to the accordion container. Useful for custom styling or layout adjustments.
Component Outputs (Events)
openHandler EventEmitter<AccordionItem> - Emitted when an accordion panel is opened or closed. Provides the AccordionItem that was toggled, including its current state. Use to track which panels are open or to perform actions when panels are toggled.

# AccordionItem Interface

The AccordionItem interface defines the structure for accordion items passed to the items input property.

Property Type Default Description
id string - Required. Unique identifier for the accordion item. Used internally to track which panel is open/closed.
title string - Required. The text displayed in the accordion header button. This is the visible label for the collapsible panel.
content string | TemplateRef<any> - Required. The content displayed when the panel is open. Can be a string (HTML) for simple content or an Angular TemplateRef for complex, reusable content with custom components.
caption string undefined Optional secondary text displayed below or alongside the title. Use for additional context or descriptions.
buttonTemplate string | TemplateRef<any> undefined Optional template for custom button content in the accordion header. Allows adding action buttons, icons, or other interactive elements alongside the title. The template receives the item as context.
context any undefined Optional context object passed to template references. Use when content or buttonTemplate is a TemplateRef to provide data that the template can access.
count string | number undefined Optional badge count displayed next to the title. Can be a number or string (e.g., '5' or '+9'). When provided, displays as a circular badge with the specified countColor.
countColor string undefined Optional CSS class for the count badge background color. Defaults to 'bg-red-500' if not specified. Use Tailwind color classes like 'bg-yellow-500', 'bg-purple-500', etc.
open boolean undefined Optional initial open state for the accordion panel. When true, the panel starts expanded. When allowMultiple is false, only one panel can be open at a time, so setting multiple items to open: true will result in only the last one being open.
