Footer

# API Reference

Property Type Default Description
Component Inputs
leftMenus FooterMenu[] [] Array of menu items displayed on the left side of the footer. Each menu item is rendered as a link with a vertical divider between items. Items are displayed in a horizontal list that wraps on smaller screens. See the FooterMenu Interface section below for details.
rightMenus FooterMenu[] [] Array of menu items displayed on the right side of the footer. Each menu item is rendered as a link with a vertical divider between items. Items are displayed in a horizontal list that wraps on smaller screens. On larger screens (3xl breakpoint and above), the left and right menus are displayed side by side. On smaller screens, they stack vertically. See the FooterMenu Interface section below for details.
className string '' Optional additional CSS classes to apply to the footer container. Useful for custom styling or layout adjustments. The footer has default styling with a top border, padding, and container queries for responsive behavior.

# FooterMenu Interface

The FooterMenu interface defines the structure for menu items passed to the leftMenus and rightMenus input properties.

Property Type Default Description
title string - Required. The display text for the menu link. Shown as the clickable link text in the footer. This is the visible label for the navigation item.
href string - Required. The URL or route path for the menu link. Used as the href attribute of the anchor tag. Can be a relative path, absolute URL, or route path depending on your routing setup.
