Actions Layout

# API Reference

Property Type Default Description
Component Inputs
items ActionItem[] [] Required. Array of action items to display in the layout. Each item represents a clickable action button with an icon. See the ActionItem Interface section below for details.
className string '' Optional additional CSS classes to apply to the actions layout container. Useful for custom styling or layout adjustments (e.g., setting max width: max-w-fit ml-auto).
selectedClass string '' Optional CSS classes to apply to selected action buttons. When an action is clicked, it becomes selected and this class is applied in addition to the default selected styling (red color). Use for additional custom styling of selected states.

# ActionItem Interface

The ActionItem interface defines the structure for action items passed to the items input property.

Property Type Default Description
iconId string - Required. The icon identifier displayed when the action is not selected. Should match an available icon in your icon library (e.g., 'eye-slash', 'bell-02', 'message-chat-01').
iconIdFilled string - Required. The icon identifier displayed when the action is selected. Should match an available filled/solid version of the icon in your icon library (e.g., 'eye-closed-filled', 'bell-filled', 'messagerie-filled'). The component automatically switches between iconId and iconIdFilled based on selection state.
handler () => void - Required. Callback function executed when the action button is clicked. Use to perform the action associated with the button (e.g., open a modal, navigate, toggle a feature).
className string undefined Optional CSS classes to apply to the individual action button. Use for custom styling of specific action items (e.g., different colors or sizes for different actions).
badge boolean undefined Optional flag to display a badge indicator on the action button. When true and the icon contains 'bell', displays a red circular badge in the top-right corner of the icon. Use to indicate notifications or unread items.
disabled boolean undefined Optional flag to disable the action button. When true, the button becomes non-interactive and may be styled differently to indicate it is disabled.
