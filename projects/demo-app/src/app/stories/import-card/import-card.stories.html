Import Card

# API Reference

Property Type Default Description
Component Inputs
title string '' The title text displayed in the card. Shown in bold text above the description. This is the primary heading for the import card (e.g., 'Virement SIMT').
description string '' The description text displayed below the title. Shown in smaller text. Use to provide additional context or details about the import option (e.g., 'Domestique').
actions ImportCardAction[] [] Array of action buttons displayed at the bottom right of the card. Each action is rendered as a button with a label and icon. Actions are displayed horizontally with spacing between them. See the ImportCardAction Interface section below for details.
icon string undefined Optional icon identifier displayed on the left side of the card (before the title and description). Should match an available icon in your icon library (e.g., 'cookie', 'file', 'upload'). When provided, the icon is displayed in a rounded container with border and background. When not provided, only the title and description are shown.
className string '' Optional additional CSS classes to apply to the import card container. Useful for custom styling or layout adjustments. The card has default styling with background, padding, border radius, and relative positioning.

# ImportCardAction Interface

The ImportCardAction interface defines the structure for action buttons passed to the actions input property.

Property Type Default Description
label string - Required. The button label text displayed on the action button. Shown in bold text next to the icon (e.g., 'Import', 'Export', 'Download').
icon string - Required. Icon identifier displayed on the right side of the button label. Should match an available icon in your icon library (e.g., 'upload-cloud', 'download', 'file'). The icon appears in primary color with a size of 16px (w-4 h-4).
onClick () => void - Required. Callback function executed when the action button is clicked. Use to handle button clicks and perform actions (e.g., trigger file import, open dialogs, navigate to pages).
