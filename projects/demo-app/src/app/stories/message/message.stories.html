Message

# API Reference

Property Type Default Description
Component Inputs
variant 'success' | 'info' | 'warning' undefined Visual style variant for the message component. Determines the color scheme, icon, and overall appearance:
  • 'warning': Amber/yellow theme with danger icon. Used for warnings or cautionary messages.
  • 'success': Green theme with check icon. Used for success messages or confirmations.
  • 'info': Blue theme with info-circle icon. Used for informational messages or general notices.
Each variant has distinct styling for the wrapper (background, border), icon container, and content text.
content string '' The message text content displayed in the component. Shown in bold text next to the icon. This is the main message that users will read. The text color and styling vary based on the selected variant.
className string '' Optional additional CSS classes to apply to the message component container. Useful for custom styling, layout adjustments, or responsive sizing. The component has default styling with rounded corners, border, padding, and flex layout. Additional classes are applied alongside the variant-specific theme classes.
