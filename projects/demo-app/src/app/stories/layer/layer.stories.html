Layer

Default Layer

A layer is a floating panel that slides in with animation. It includes a close button and supports content projection.

Open Layer

This is the layer content. Click the X button to close.

# API Reference

Property Type Default Description
Component Inputs
isOpen boolean false Controls the visibility of the layer. When true, the layer is displayed with a slide-in animation. When false, the layer is hidden with a slide-out animation. Supports two-way binding with [(isOpen)].
className string '' Optional additional CSS classes to apply to the layer container. Useful for custom styling, sizing, or positioning adjustments.
Component Outputs (Events)
isOpenChange EventEmitter<boolean> - Emitted when the layer's open state changes. Primarily used for two-way binding with [(isOpen)]. Emits false when the layer is closed via the close button.

# Features

Feature Description
Animation The layer uses a smooth slide-in/slide-out animation with fade effect. Enter animation: 200ms ease-out from top. Leave animation: 200ms ease-in to top.
Close Button Built-in close button (X icon) positioned at the top-right corner. Clicking it emits isOpenChange with false value.
Content Projection Supports ng-content for flexible content. Any content placed between <ds-layer> tags will be projected into the layer body.
Styling Default styling includes bg-level-3 background, rounded-lg corners, and p-6 padding. Uses relative positioning for proper close button placement.

# Usage Examples

Below are common usage patterns for the Layer component.

Pattern Example
Basic Two-Way Binding <ds-layer [(isOpen)]="isLayerOpen">
  <p>Layer content here</p>
</ds-layer>
With Custom Classes <ds-layer [(isOpen)]="isOpen" className="max-w-md mx-auto">
  <p>Centered layer</p>
</ds-layer>
With Event Handling <ds-layer [isOpen]="isOpen" (isOpenChange)="onLayerClose($event)">
  <p>Layer content</p>
</ds-layer>
